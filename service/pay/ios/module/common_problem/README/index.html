常见问题

Q:请问你们的支付sdk怎么收费 A:渠道费和公司税共收取8%。其中渠道费是支付宝/微信收取的费用。

Q:为什么我支付了0.01但后台看到只有0.00 A:请勿担心，在Bmob财务管理平台订单管理处，金额从小数点后第三位开始不显示，比如支付了0.01元实收0.00，其实是0.0095。
